# Bonus features

* **Error** messages are colored with red, while **Warning** messages are colored with purple

## Error

* PASS_FUNCTION_TO_SCALAR
* IS_FUNCTION_NOT_VARIABLE
* VOID_VARIABLE
* TYPEDEF_VOID_ARRAY
* PARAMETER_TYPE_UNMATCH
* PASS_VOID_TO_SCALAR
* UNARY_VOID
* FUNCTION_NOT_RVALUE
* RETURN_ARRAY

## Warning

* DIVIDE_BY_ZERO
* NON_VOID_RETURN_VOID
* VOID_RETURN_NON_VOID

## Optimization

* Constant Folding
